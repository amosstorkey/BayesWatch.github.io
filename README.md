# BayesWatch.github.io
Group homepage

## Adding a Blog post

add a markdown file to https://github.com/BayesWatch/BayesWatch.github.io/_posts with the name `<YYYY>-<MM>-<DD>-<SHORT_TITLE>.md`.

The contents of the file should be:

```
---
layout: post
title: <YOUR TITLE>
author: <FIRSTNAME> <SECONDNAME>
---
<CONTENTS>
```
